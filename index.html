About Me

I am passionate about the pursuit of knowledge and the joy of designing what is essentially thought-stuff. Software is little more than imagination, but it is something which also provides real utility.

The programmer, like the poet, works only slightly removed from pure thought-stuff. He builds his castles in the air, from air, creating by exertion of the imagination. Few media of creation are so flexible, so easy to polish and rework, so readily capable of realizing grand conceptual structures.

— The Mythical Man Month by Frederic P. Brooks

I believe that by learning new ideas, methodologies and tools, a designer gains new instruments which she can use to sculpt grander castles in the sandbox of her imagination.

Experience

I am an experienced software developer at Strata Health where I design and implement solutions used by clinicians in patient flow management. In 2009, I completed my BSc in Computer Science with an internship at IBM Canada where I constructed and verified interfaces used to expose database monitoring information.

Expertise

My expertise is in web design and web application development which encompases enabling technologies: frontend/backend technologies including RDBMSs, languages and frameworks; web architecture and W3C specifications.

I believe that a broad and detailed field of expertise entails familiarity with the right tool for any problem. For example: my server-side languages of choice are Java and PHP, but using JavaScript with Node.js when creating RESTful JSON services is just too simple; my favourite RDBMS is MySQL, but complex requirements demand a feature-rich RDBMS such as Oracle or DB2; similarly, REST and JSON is simpler to implement and consume (especially with AJAX) but SOAP and XML is arguably more flexible and has richer standards.

Interests

In terms of web technologies, my interests currently include: HTML 5 (particularly in creating mobile offline web apps) and CSS 3; Ruby on Rails; NoSQL database systems such as MongoDB; XML-related particularly XSL and XQuery; semantic web, social media and online user experience.
